C-lau
=====

C-lau (spelled `cờ lau` - which may remind you of [Đinh Bộ Lĩnh](http://vi.wikipedia.org/wiki/%C4%90inh_Ti%C3%AAn_Ho%C3%A0ng))
is A Linux tray app that helps normalize Vietnamese diacritics from clipboard.
